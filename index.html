Profile

I am a software developer and bioinformatician. For the last 20 years I have been helping biologists to capture, analyse and interpret genomic data.

Skills and Experience

  • Software development: web, desktop and server
  • Database administration and maintenance
  • Genome assembly, annotation and analysis
  • Processing, management and analysis of next generation sequencing data
  • Systems administration including software and database installation and configuration
